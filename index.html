Temas que hemos visto

Condicionales

Las condicionales permiten tomar decisiones dependiendo de si una condición es verdadera o falsa. 


let nota = 4;

if (nota >= 3) {
    console.log("Aprobó");
} else {
    console.log("Perdió");
}

Variables

Las variables sirven para guardar información. 


let nombre = "Carlos";
let edad = 16;

Ciclo While

El ciclo while repite código mientras una condición se cumpla. 


let numero = 1;

while (numero <= 5) {
    console.log(numero);
    numero++;
}

Switch

Switch permite evaluar diferentes casos. 


let dia = 1;

switch (dia) {

    case 1:
        console.log("Lunes");
        break;

    case 2:
        console.log("Martes");
        break;

    default:
        console.log("Otro día");
}
